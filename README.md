# go-gqlgen-gin-template

## Add module requirements and sums

```
go mod tidy
```

## Edit ths db config file

- Config file : `./db/dbconfig.ini`

```
host     = "localhost"
database = "postgres"
user     = "postgres"
port     = "5432"
password = "password"
```

## run the server
```
go run server.go
```
